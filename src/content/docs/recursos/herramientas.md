## Dentro del Juego

### El Sistema de Replays
- **Dónde:** Menu Principal → Replays
- **Qué podés ver:** Todo: recursos de ambos jugadores, posición de unidades, timing de edificios
- **Cómo usarlo:** Ver tus replays en 4x speed para el macro, 1x para momentos críticos
- Ver la guía completa: [Análisis de Replays](../04-nivel-experto/04-analisis-de-replays-pro)

### El Post-Game Screen
- Después de cada partida: estadísticas de recursos recolectados, unidades producidas, tiempo de cada edad
- **Lo más útil:** El gráfico de recursos muestra si tuviste cuellos de botella

### Modo Espectador / Observer
- Podés observar partidas de otros jugadores en el lobby
- **Tip:** Buscar partidas de jugadores de 200-400 ELO más que vos para estudiar su macro

## Sitios Web Esenciales

### aoestats.io
- [aoestats.io/maps/arabia/](https://aoestats.io/maps/arabia/)
- Win rates por civ y mapa en tiempo real
- **Uso:** Verificar el meta actual antes de elegir civ

### aoe2.gg
- [aoe2.gg](https://aoe2.gg)
- Estadísticas de partidas, historial de jugadores, perfiles
- **Uso:** Ver el historial de partidas y win rate por civ de cualquier jugador (incluyendo vos)

### aoecompanion.com
- [aoecompanion.com](https://aoecompanion.com)
- Build guides + estadísticas integradas
- **Uso:** Seguir build orders con timer mientras jugás

### aoe2.net (aoe2insights)
- [aoe2insights.com](https://www.aoe2insights.com/)
- Estadísticas de meta, tendencias
- **Uso:** Análisis de partidas pasadas, tendencias del ladder

## Herramientas de Build Order

### Build Order Guide
- [buildorderguide.com](https://www.buildorderguide.com/)
- Build orders interactivos con timer en tiempo real
- Podés abrirlo en segundo monitor mientras jugás
- **Cómo usarlo:** Seleccionás el build (ej: "21 pop scout rush"), activás el timer al empezar la partida, y te va diciendo qué hacer en cada segundo

### AOE Builds
- [aoebuilds.com](https://www.aoebuilds.com/)
- Lista completa de builds en texto
- Includes: Scouts Pop 20, Archers Pop 19, MAA, FC, y variaciones

### Hera's Build Orders (PDF actualizable)
- Disponible en su Discord oficial y en ageofnotes.com
- **URL referencia:** [ageofnotes.com/beginners/hera-strategy-guide-2025/](https://ageofnotes.com/beginners/hera-strategy-guide-2025/)
- 24 build orders del meta 2025, actualizados regularmente

## Apps y Overlays

### AoE2 Companion App
- Disponible para Windows
- Overlay que muestra stats de la partida en tiempo real
- **Función:** APM counter, recursos por segundo, alertas de aldeano inactivo

### AoE2 Insights
- [aoe2insights.com](https://www.aoe2insights.com/)
- Análisis post-partida detallado
- Muestra el TC idle time, distribución de aldeanos por minuto, comparativa con el oponente

## Cómo Configurar un Setup de Estudio Óptimo

### Configuración recomendada (2 monitores):
- **Monitor 1:** AoE2 a resolución máxima
- **Monitor 2:** Build order guide o replay de referencia

### Solo 1 monitor:
- Mantené el build order en papel al lado del teclado
- Usá Alt+Tab para checar aoestats antes de iniciar partida

### El "Study Session":
1. Antes de la partida: checar el meta (aoestats), elegir civ y build
2. Durante: ejecutar el build con objetivos específicos
3. Después: 10-15 min de replay para identificar el error más costoso

## Hotkey Management

### Importar Hotkeys de Pros
- Hera y otros pros comparten sus archivos de hotkeys en su Discord
- El archivo se importa desde: Opciones → Hotkeys → Import Profile
- **Recomendación:** Empezar con el preset "Grid" del juego, luego personalizar

### La Configuración Grid (recomendada para todos)
- Las teclas reflejan la posición física de los botones en la interfaz
- QWER para fila superior de acciones, ASDF para segunda fila, ZXCV para tercera
- Reducción masiva de tiempo de mano→teclado

### Age of Notes

- [ageofempires.fandom.com](https://ageofempires.fandom.com)
- Stats exactos de todas las unidades y techs, actualizados con cada parche
- **Uso:** Verificar números exactos (HP, ataque, costos actuales)
