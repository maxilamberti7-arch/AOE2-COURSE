---

## Principios Generales

1. **Primero ataque, luego armadura:** +1 ataque = menos golpes para matar. +1 armadura = 1 golpe más para morir. El ataque escala mejor.
2. **No investigues para unidades que no tenés:** Escale Barding Armor sin knights = desperdicio.
3. **Las techs económicas baratas tienen ROI rápido:** Double-Bit Axe (100F) se paga en 5 minutos. No postergues las económicas.
4. **Prioridad al upgrade de unidad principal:** Si jugás archers, Crossbow > todos los upgrades de blacksmith hasta que esté listo.

## Orden por Build Order

### Scout Rush → Knights → Paladin

**Feudal (en orden):**
1. Bloodlines (150F + 100G) — scouts más resistentes inmediatamente
2. Double-Bit Axe (100F) — eco
3. Horse Collar (75F + 75W) — eco
4. Forging (150F) — +1 ataque a los scouts/knights futuros

**Castle Age (en orden):**
1. Cavalier upgrade (300F + 300G) si tenés knights
2. Chain Barding Armor (250F + 100G)
3. Iron Casting (220F + 120G)
4. Husbandry (150F) — si los knights necesitan velocidad para kitar
5. Bow Saw (100F + 150W), Heavy Plow (125F + 125W)
6. Gold Shaft Mining (200F + 100W) — si tenés 4+ en oro
7. Chivalry (unique tech Franks) si jugás Franks

**Imperial Age (en orden):**
1. Paladin upgrade (1300F + 750G) — si está disponible para tu civ
2. Blast Furnace (275F + 225G)
3. Plate Barding (350F + 150G)
4. Hand Cart (300F + 200W)
5. Crop Rotation (250F + 250W)

### Archer Rush → Arbalest

**Feudal (en orden):**
1. Fletching (100F + 100G) — +1 ataque inmediato a los archers
2. Double-Bit Axe (100F)
3. Horse Collar (75F + 75W)
4. Padded Archer Armor (100F) — armadura para los archers
5. Gold Mining (100F + 75W) — si tenés 3+ en oro

**Castle Age (en orden):**
1. Crossbow upgrade (150F + 100W) — EL upgrade más importante de este BO
2. Ballistics (300F + 175W) — OBLIGATORIO para archers
3. Thumb Ring (300F + 150W) — OBLIGATORIO
4. Bodkin Arrow (200F + 150G) — +1 ataque más
5. Leather Archer Armor (150F + 150G)
6. Heavy Plow + Bow Saw — eco

**Imperial Age:**
1. Arbalest upgrade (1000F + 600G)
2. Bracer (300F + 200G) — +1 ataque + +1 rango
3. Ring Archer Armor (250F + 250G)
4. Blast Furnace (para Skirmisher soporte si los usás)
5. Hand Cart, Crop Rotation

### Fast Castle Boom → Multi-TC

**Castle Age (boom prioritized):**
1. Segundo TC, tercer TC — primero que todo lo demás
2. Hand Cart (300F + 200W) — con 2+ TCs, el beneficio es masivo
3. Heavy Plow (125F + 125W)
4. Bow Saw (100F + 150W)
5. Gold Shaft Mining (200F + 100W)
6. Entonces: upgrades militares según la composición que estés construyendo

## El Error de la "Tech Queue" Larga

Un error común en 1100-1300 ELO: hacer cola de 5-6 investigaciones seguidas. El problema:

- Con 1 solo Blacksmith, investigar 5 techs tarda 5+ minutos
- Durante esos minutos, tus unidades están underperforming porque no tienen los upgrades
- Si una tech tarda 60 segundos y hacés 5 en cola: tus unidades pasan 5 minutos sin mejoras

**Solución:** Priorizá la 1-2 más impactantes AHORA. Podés investigar el resto después.

## Tecnologías que Casi Nunca Vale la Pena Investigar en 1v1 Arabia

| Tech | Por qué evitar |
|------|---------------|
| Guilds | Solo útil en TG con comercio activo |
| Caravan | Solo en TG |
| Spy / Treason | El costo es tu población × 30 gold → carísimo |
| Banking | Situacional (solo si usás el Mercado mucho) |
| Fortified Wall | Raro que valga vs Trebs que la destruyen igual |
| Bombard Tower | Muy caro en stone para el beneficio |
| Sappers | +15 attack infantry vs buildings, pero el costo es alto para uso nicho |
